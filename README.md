# Alamgeer Shekh Portfolio

A personal portfolio website built with React and Vite to showcase my profile, technical skills, projects, resume, and contact details.

## Features

- Multi-page portfolio structure
- Light and dark mode toggle
- Developer profile section with photo
- Skills page covering frontend, backend, databases, tools, and languages
- Projects page with portfolio-ready project cards
- Resume page with direct PDF access
- Contact page with quick links
- Responsive layout for desktop and mobile

## Tech Stack

- React.js
- Vite
- JavaScript
- CSS
- Tailwind CSS (package included in setup)

## Projects Included

- Portfolio Website
- HealthMate Go
- Tech Path Finder
- Blogging Web Application

## Local Setup

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm run dev
```

3. Build for production:

```bash
npm run build
```

4. Preview the production build:

```bash
npm run preview
```
